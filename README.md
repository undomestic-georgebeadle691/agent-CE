## üìö Overview

Agent-CE is a platform designed to help you evaluate web browsing agents easily. With Agent-CE, you can run various agent frameworks such as Browser Use, Notte, Anthropic Computer Use, and OpenAI Computer Use. It offers production-ready Docker images and efficient CI/CD pipelines, making it simple to get started.

## üíª System Requirements

To run Agent-CE effectively, ensure your system meets the following requirements:

- **Operating System:** Windows 10 or later, macOS 10.14 or later, or a modern Linux distribution.
- **RAM:** 8 GB minimum (16 GB recommended).
- **Processor:** Intel i5 or equivalent.
- **Docker:** Docker Desktop installed (latest version recommended).
- **Disk Space:** At least 2 GB of free disk space.
